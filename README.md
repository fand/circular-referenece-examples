# circular-reference-examples

This repository shows how circular-references are handled in following environment:

- Node.js + CommonJS
- Webpack + CommonJS
- Webpack + ES Modules
- Webpack + TypeScript + ts-loader
- Browserify + CommonJS
- Browserify + ES Modules + babelify

## Result

```
$ node node/index.js
Node.js undefined
```
